## Install 

Run the geth console first.
{% highlight markdown %}
npm install -g @vue/cli
# OR
yarn global add @vue/cli
{% endhighlight %}

## Create a project
{% highlight markdown %}
vue create my-project
# OR
vue ui

{% endhighlight %}
